About

This is a game based on the board game Articulate! for revising the GCSE OCR computer science currciculum.

Rules

After you press Start, a pop-up will appear with a word on it. Describe the word to your partner within a time limit. (You decide.) If they get it right, press OK without typing anything. If you want to pass, type p and press OK. If the time limit has run out, type s and press OK. If you want to quit, press Cancel.
